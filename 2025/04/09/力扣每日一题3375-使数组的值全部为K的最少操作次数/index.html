力扣每日一题3375-使数组的值全部为K的最少操作次数

2025-04-09 做题笔记 191 字
cpp leetcode

题意

数h对于数组nums,若满足,则称h合法，即数组中大于h的数均相等。

每次操作可以选择任意合法h使得nums中所有大于h的元素变为h

求问将nums中所有数变为给定k值所需最少操作次数。

思路

当nums最小值小于k时显然是不存在合法操作的，因为不存在将元素值变大的操作。

可以发现每次操作使得最大值变为次大值时有最优，此题本质可转化为数组中大于k的元素种类数。

实现

class Solution {
public:
    int minOperations(vector<int>& nums, int k) {
        ranges::sort(nums);
        if (nums.front() < k) return -1;
        nums.erase(unique(nums.begin(), nums.end()), nums.end());
        return nums.size() - (nums.front() == k);
    }
};
