力扣每日一题2845-统计趣味子数组的数目

2025-04-25 做题笔记 218 字
cpp leetcode

题意

给定整数数组nums，以及整数modulo和k。

记数组内满足x % modulo == k的个数为cnt

若nums的某子数组满足cnt % modulo == k则称其为趣味子数组，求nums内的趣味子数组个数。

思路

首先可以发现我们仅关注满足x % modulo == k的x的个数，因此我们可以将所有nums内模modulo为k的数记为1其他数则记为0;

完成转换后什么样的子数组满足cnt % modulo == k呢，显然需要满足

那么是不是可以联想到前缀和，再结合一下模运算的性质，可以得出结果。

实现

class Solution {
using ll = long long;
public:
    long long countInterestingSubarrays(vector<int>& nums, int modulo, int k) {
        ll ans = 0;
        unordered_map<int, int> cnt;
        int pre = 0;
        for (const auto& x : nums) {
            cnt[pre]++;
            pre += x % modulo == k;
            if (pre >= modulo) pre -= modulo;
            ans += cnt[(pre - k + modulo) % modulo];
        }
        return ans;
    }
};
